## 🚀 部署步骤

### 1. 构建项目
```bash
cd frontend
npm run build
```

### 2. 上传文件到服务器
将整个 `dist/` 目录上传到您的服务器web根目录

### 3. 服务器配置要求
- 支持静态文件服务
- 支持HTTPS
- 支持现代JavaScript (ES6+)
- 至少1.5GB可用磁盘空间

## ⚠️ 重要提醒

1. **不要修改文件路径**: 所有文件的相对路径必须保持不变
2. **不要遗漏文件**: 缺少任何文件都可能导致功能异常
3. **确保文件完整性**: 上传过程中确保文件没有损坏
4. **服务器配置**: 确保服务器MIME类型配置正确
