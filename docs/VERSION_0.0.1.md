# 邹韬奋纪念网页 v0.0.1

## 版本信息
- **版本号**: 0.0.1
- **发布日期**: 2025-07-30
- **状态**: MVP (最小可行产品) - 前端独立版本

## 项目概述

邹韬奋纪念网页是一个展示中国著名爱国者和新闻出版先驱邹韬奋先生生平、思想和贡献的现代化Web应用。本版本专注于前端展示功能，提供完整的用户界面和交互体验。

## 技术栈

### 前端技术
- **React 18** - 现代UI框架

### 开发工具

## 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn

### 安装和运行

1. **安装依赖**:
   ```bash
   cd frontend
   npm install
   ```

2. **启动开发服务器**:
   ```bash
   npm run dev
   ```
   应用将在 `http://localhost:3000` 启动

3. **构建生产版本**:
   ```bash
   npm run build
   ```

4. **预览生产版本**:
   ```bash
   npm run preview
   ```

### 其他命令
- `npm run lint` - 代码检查
- `npm run test` - 运行测试
- `npm run test:coverage` - 生成测试覆盖率报告
