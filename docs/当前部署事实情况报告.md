## 🐳 Docker配置详细分析

### 📋 Cantaloupe容器配置
```yaml
image: uclalibrary/cantaloupe:5.0.7-0
container_name: cantaloupe  
restart: unless-stopped
ports: "127.0.0.1:8282:8182"        # 仅本地访问
volumes:
  - /srv/iiif:/imageroot:ro          # IIIF数据目录 (只读)
  - /srv/iiif/cache:/var/cache/cantaloupe
  - /srv/iiif/logs:/var/log/cantaloupe
```

**关键配置**:
- **图像路径**: `/srv/iiif/images/` → 容器内 `/imageroot/images/` (🗞️ 仅报纸刊物)
- **JVM内存**: 512MB-1GB (适合2GB服务器)
- **API访问**: api用户 + 管理员admin (密码: nsrjd4yt)
- **缓存**: 禁用服务器端缓存 (节省内存)

### 📋 Caddy容器配置  
```yaml
image: caddy:2
container_name: caddy
restart: unless-stopped  
ports: ["80:80", "443:443"]          # 公网访问
volumes:
  - ./Caddyfile:/etc/caddy/Caddyfile:ro
  - /srv/iiif/manifests:/srv/iiif/manifests:ro
```

**关键功能**:
- **SSL终止**: 自动HTTPS证书管理
- **反向代理**: 路由IIIF请求到Cantaloupe (8282端口)
- **静态文件**: 直接服务清单文件 (`/srv/iiif/manifests/`)
