# 瀑布流图片缓存实现文档

## 概述

本文档详细描述了为 Taofen Library 项目瀑布流布局实现的图片缓存系统。该系统扩展了现有的 IIIF 缓存服务，为手写体图片提供完整的缓存解决方案。

## 系统架构

### 缓存层次结构

```
浏览器层 (Client-side)
├── 内存缓存 (HandwritingMemoryCache)
├── 图片预加载 (ImagePreloader)
└── Redis缓存 (通过API)

服务端层 (Server-side)
├── Redis缓存服务 (cache-service)
├── 图片元数据缓存
├── 图片尺寸缓存
├── 布局计算缓存
└── 预加载队列缓存
```

### 核心组件

#### 1. 服务端组件

**ImageCache** (`cache-service/lib/image-cache.js`)
- 图片元数据缓存
- 图片尺寸缓存
- 瀑布流布局缓存
- 预加载队列缓存
- 批量操作支持

**API接口** (`cache-service/server.js`)
- `POST /api/cache/image/metadata` - 缓存图片元数据
- `GET /api/cache/image/metadata/:imageId` - 获取图片元数据
- `POST /api/cache/image/dimensions` - 缓存图片尺寸
- `GET /api/cache/image/dimensions/:imageId` - 获取图片尺寸
- `POST /api/cache/image/dimensions/batch` - 批量获取图片尺寸
- `POST /api/cache/image/layout` - 缓存瀑布流布局
- `GET /api/cache/image/layout/:layoutKey` - 获取瀑布流布局
- `POST /api/cache/image/preload` - 缓存预加载队列
- `GET /api/cache/image/preload/:queueId` - 获取预加载队列
- `DELETE /api/cache/image/:imageId` - 清除图片缓存
- `GET /api/cache/image/stats` - 获取图片缓存统计

#### 2. 前端组件

**ImageCacheService** (`frontend/src/services/cache/image-cache-service.ts`)
- 统一的缓存服务接口
- TypeScript 类型安全
- 智能缓存策略
- 错误处理和重试机制

**React Hooks** (`frontend/src/hooks/useImageCache.ts`)
- `useImageDimensions` - 图片尺寸缓存管理
- `useLayoutCache` - 瀑布流布局缓存管理
- `usePreloadQueue` - 预加载队列管理
- `useImageCacheStats` - 缓存统计信息
- `useSmartImagePreloader` - 智能图片预加载
- `useImageCacheHealth` - 缓存健康检查

## 使用方法

### 1. 在组件中使用图片缓存

```tsx
import { useImageDimensions, useLayoutCache } from '@/hooks/useImageCache';

function HandwritingMasonryGrid({ items, columns, viewport }) {
  // 图片尺寸缓存
  const { dimensions, cacheDimensions } = useImageDimensions(
    items.map(item => item.image)
  );

  // 布局缓存
  const { layout, cacheLayout } = useLayoutCache(
    items.map(item => item.id),
    columns,
    viewport
  );

  // 当布局计算完成时缓存结果
  const handleLayoutCalculated = useCallback((calculatedLayout) => {
    cacheLayout(calculatedLayout);
  }, [cacheLayout]);

  return (
    <div className="masonry-grid">
      {items.map((item, index) => (
        <HandwritingCard
          key={item.id}
          item={item}
          dimensions={dimensions[item.image] || item.dimensions}
        />
      ))}
    </div>
  );
}
```

### 2. 智能预加载使用

```tsx
import { useSmartImagePreloader } from '@/hooks/useImageCache';

function HandwritingModule() {
  const { currentPageItems, nextPageItems } = useHandwritingData();
  
  // 智能预加载
  const { loading, loadedImages } = useSmartImagePreloader(
    currentPageItems.map(item => item.image),
    nextPageItems.map(item => item.image),
    {
      enabled: true,
      priority: 'high',
      concurrency: 3
    }
  );

  return (
    <div>
      {loading && <div>预加载中...</div>}
      <HandwritingMasonryGrid items={currentPageItems} />
    </div>
  );
}
```

### 3. 缓存统计和监控

```tsx
import { useImageCacheStats, useImageCacheHealth } from '@/hooks/useImageCache';

function CacheMonitor() {
  const { stats, loading: statsLoading } = useImageCacheStats();
  const { isHealthy, lastCheck } = useImageCacheHealth();

  return (
    <div className="cache-monitor">
      <div>缓存状态: {isHealthy ? '健康' : '异常'}</div>
      <div>最后检查: {lastCheck?.toLocaleString()}</div>
      {stats && (
        <div>
          <div>缓存命中率: {stats.hitRate}%</div>
          <div>内存使用: {stats.memoryUsage}MB</div>
          <div>缓存条目: {stats.totalKeys}</div>
        </div>
      )}
    </div>
  );
}
```

### 4. 扩展数据获取Hook

```tsx
import { useHandwritingData } from '@/hooks/useHandwritingData';

function HandwritingPage() {
  const {
    handwritingItems,
    loading,
    error,
    prefetchImageDimensions,
    clearImageCache
  } = useHandwritingData();

  // 预获取图片尺寸
  useEffect(() => {
    if (handwritingItems.length > 0) {
      const imageUrls = handwritingItems.map(item => item.image);
      prefetchImageDimensions(imageUrls);
    }
  }, [handwritingItems, prefetchImageDimensions]);

  const handleClearCache = async () => {
    await clearImageCache();
    // 重新加载数据
    window.location.reload();
  };

  return (
    <div>
      <button onClick={handleClearCache}>清除缓存</button>
      <HandwritingMasonryGrid items={handwritingItems} />
    </div>
  );
}
```

## 缓存策略配置

### TTL配置

```typescript
export const IMAGE_CACHE_TTL = {
  // 图片元数据 - 24小时
  METADATA: 86400,
  
  // 图片尺寸 - 30天（相对稳定）
  DIMENSIONS: 2592000,
  
  // 瀑布流布局 - 5分钟（频繁变化）
  LAYOUT: 300,
  
  // 预加载队列 - 1分钟（临时数据）
  PRELOAD_QUEUE: 60,
};
```

### 缓存键生成规则

```typescript
// 图片元数据: image:meta:{imageId}
// 图片尺寸: image:dimensions:{imageId}

// 预加载队列: image:preload:{queueId}
```
