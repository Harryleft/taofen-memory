## 🏗️ 技术实现

### 核心文件结构
```
frontend/
├── public/
│   └── uv_simple.html              # 独立的UV查看器页面
└── src/
    └── components/
        └── newspapers/
            ├── services.ts           # 重写的数据服务
            ├── NewspapersModule.tsx   # 重写的主模块组件
            ├── ViewerPage.tsx        # 重写的查看器页面
            └── iiifTypes.ts          # IIIF数据类型定义
```

### 关键技术特点

#### 1. uv_simple.html
- 独立的UV查看器页面
- 从URL hash参数获取manifest ID
- 支持嵌入式模式
- 完善的错误处理和重新加载功能

#### 2. NewspaperService
- 支持代理模式处理CORS
- 异步加载刊物和期数信息
- 搜索和过滤功能
- URL编码处理

#### 3. NewspapersModule
- 双视图状态管理
- 防抖搜索功能
- 多种排序方式
- 响应式布局

#### 4. ViewerPage
- 基于iframe的UV加载
- 时间戳参数避免缓存问题
- 消息监听和错误恢复
- 安全的sandbox属性

## 🚀 使用流程

```
访问 /bookstore-timeline
    ↓
点击"数字报刊"标签
    ↓
使用搜索框搜索刊物（可选）
    ↓
使用排序功能排列刊物（可选）
    ↓
选择刊物查看期数列表
    ↓
点击期数进入UV查看器
    ↓
使用UV查看器浏览报刊
    ↓
点击"返回期刊目录"返回
```
